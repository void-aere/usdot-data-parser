## Average annual aircraft flight hour calculator

This program calculates the average airborne hours per aircraft for all
available years reported in USDOT's Form 41 data.

To run the program, first compile `main.cpp`.

You may also have to download the dataset; follow the
[instructions](data/README.md) in `data/`.
